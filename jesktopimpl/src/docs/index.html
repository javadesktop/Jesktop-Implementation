What is it?

This is the reference implementation of the  Jesktop API project 

Jesktop Binary Download

Jesktop can be downloaded from here in two forms.

Jesktop zip file

Take the Jesktop zip file, unzip it, execute "run". Then open the "Simple Installer" tool and paste in a link from this page Jesktop apps This should hot install applications.

Jesktop Source Download

The usual two forms : zip and tar.gz

Application Downloads

Applications for Jesktop are listed on the jesktopapps.sourceforge.net website. There are a couple of applications that come with Jesktop from the Apache site - "Explorer" and some miscellaenous apps.
